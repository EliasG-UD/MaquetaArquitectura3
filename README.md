# Integra PYME Fácil — Maqueta (GitHub Pages)

**Maqueta estática de la aplicación** (sin backend ni base de datos). Incluye navegación SPA con `hash`, tablas con datos ficticios y formularios de demostración.

## Estructura
```
.
├─ index.html
└─ assets/
   ├─ styles.css
   ├─ app.js
   ├─ mock-data.js
   └─ logo.svg
```

## Cómo desplegar en GitHub Pages
1) Crea un repositorio (p. ej. `integra-pyme-facil-mock`).  
2) Sube estos archivos a la rama `main`.  
3) Ve a **Settings → Pages** y selecciona **Deploy from a branch**, luego `main` y carpeta **root**.  
4) Abre la URL `https://TU-USUARIO.github.io/integra-pyme-facil-mock/`.

> Nota: Esta maqueta no guarda datos ni se conecta a ningún servicio. Todos los cambios se pierden al recargar.
